## 🚀 快速开始

### 安装依赖

```bash
# Python 3.8+
pip install pandas numpy scipy matplotlib tushare akshare

# 设置环境变量
export TUSHARE_TOKEN="your_token_here"
```

### 运行选股

```bash
# 运行选股（默认：多因子策略）
python scripts/stock_picker.py

# 指定策略
python scripts/stock_picker.py --strategy momentum    # 动量策略
python scripts/stock_picker.py --strategy value       # 价值策略
python scripts/stock_picker.py --strategy break_through  # 突破策略
```

### 回测

```bash
# 运行回测
python scripts/backtest.py --strategy momentum --start 20240101 --end 20241231
```
